#  DenWeibo框架说明
## 文件夹目录说明：
    .
    ├── DenWeibo
    │       ├── Resources       放置静态资源文件(如:font/.mp3/.mp4等)
    │       ├── General     通用文件夹
    │       │       ├── Base        基础类（Controller/model/view）
    │       │       │  └──  Controllers
    │       │       ├── Config      配置项
    │       │       ├── Tool        工具类
    │       │       │   └──  Extension
    │       │       └── Vender      第三方库
    │       ├── Modules     存放模块[如:登录、首页、发现、我的等]
    │       │       ├── Main        根节点控制器
    │       │       ├── Visitor         访客控制器
    │       │       ├── Home        首页
    │       │       │       └── Controller
    │       │       ├── Message         消息
    │       │       ├── Discover        发现
    │       │       └── Me         我
    │       ├── AppDelegate      AppDelegate.swift
    │       └── Supporting Files
    │               ├── Localizations       本地化/国际化
    │               ├── Info.plist
    │               └── Assets.xcassets     图标资源
    │..
